## 4. 给项目配置的直接建议

如果你要把这份能力直接映射到前后端配置，建议至少按模型维护这些字段：

1. `supportsModes`: `text_to_video` / `image_first` / `image_first_last` / `multimodal` / `edit` / `extend`
2. `supportedResolutions`: `['480p','720p','1080p']` 等
3. `durationRange`: 例如 `[4,15]`
4. `supportsFlexTier`: `true/false`
5. `defaultRateLimitProfile`: `enterprise_personal_split` 或 `shared_default`

这样可以避免把 2.0 的能力错误套到 1.0，或者把 1.0 pro fast 的限制漏掉。
